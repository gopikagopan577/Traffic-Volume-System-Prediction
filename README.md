#  Traffic Volume Prediction System

##  Project Overview
An end-to-end Machine Learning application that predicts hourly traffic volume using weather and time-based features.

##  Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Random Forest

##  Features
- Automated date feature extraction
- Holiday detection logic
- Weather-based inputs
- ML Pipeline with ColumnTransformer
- Feature importance visualization
- Interactive Streamlit UI

##  How to Run
1. Clone this repository
2. Install requirements: pip install -r requirements.txt
3. Run: streamlit run app.py
